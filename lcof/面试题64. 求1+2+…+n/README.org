求 =1+2+...+n=，要求不能使用乘除法、for、while、if、else、switch、case
等关键字及条件判断语句（A?B:C）。

*示例 1：*

#+begin_example
  输入: n = 3
  输出: 6
#+end_example

*示例 2：*

#+begin_example
  输入: n = 9
  输出: 45
#+end_example

*限制：*

- =1 <= n <= 10000=

** 解法
   :PROPERTIES:
   :CUSTOM_ID: 解法
   :END:
递归，结合*逻辑与*短路运算符求解。

*** *Python3*
    :PROPERTIES:
    :CUSTOM_ID: python3
    :END:
#+begin_src python
  class Solution:
      def sumNums(self, n: int) -> int:
          return n and (n + self.sumNums(n - 1))
#+end_src

*** *Java*

    :CUSTOM_ID: java

#+begin_src java
  class Solution {
      public int sumNums(int n) {
          int s = n;
          boolean t = n > 0 && (s += sumNums(n - 1)) > 0;
          return s;
      }
  }
#+end_src
