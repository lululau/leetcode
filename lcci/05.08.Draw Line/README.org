绘制直线。有个单色屏幕存储在一个一维数组中，使得32个连续像素可以存放在一个
int 里。屏幕宽度为w，且w可被32整除（即一个 int
不会分布在两行上），屏幕高度可由数组长度及屏幕宽度推算得出。请实现一个函数，绘制从点(x1,
y)到点(x2, y)的水平线。

#+end_html

x1（比特为单位）、直线结束位置
x2（比特为单位）、直线所在行数 y。返回绘制过后的数组。
