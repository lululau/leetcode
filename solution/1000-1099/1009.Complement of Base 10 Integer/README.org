每个非负整数 N 都有其二进制表示。例如， 5 可以被表示为二进制 "101"，11
可以用二进制 "1011" 表示，依此类推。注意，除 N =
0 外，任何二进制表示中都不含前导零。

  </p>
#+end_html

二进制的反码表示是将每个 1 改为 0 且每个 0 变为 1。例如，二进制数 "101" 的二进制反码为 "010"。

  </p>
#+end_html
