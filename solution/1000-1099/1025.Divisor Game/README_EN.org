that player makes a move consisting of:

#+end_html

#+end_html

#+begin_html
  <li>
#+end_html

#+begin_html
  </li>
#+end_html

#+begin_html
  <li>
#+end_html

Replacing the number n on the chalkboard with n - x.

#+begin_html
  </li>
#+end_html

#+begin_html
  </ul>
#+end_html

#+begin_html

#+end_html

Also, if a player cannot make a move, they lose the game.

#+begin_html
  </p>
#+end_html

play optimally.
