给你一个整数数组 A，只有可以将其划分为三个和相等的非空部分时才返回 true，否则返回
false。

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

形式上，如果可以找出索引 i+1 < j 且满足 A[0] + A[1] + ... + A[i] ==
A[i+1] + A[i+2] + ... + A[j-1] == A[j] + A[j-1] + ... + A[A.length -
1] 就可以将数组三等分。

  </p>
#+end_html
