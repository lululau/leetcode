给定 N，想象一个凸 N 边多边形，其顶点按顺时针顺序依次标记为 A[0], A[i],
..., A[N-1]。

  </p>
#+end_html

#+begin_html
  <p>
#+end_html

假设您将多边形剖分为 N-2
个三角形。对于每个三角形，该三角形的值是顶点标记的乘积，三角剖分的分数是进行三角剖分后所有
N-2 个三角形的值之和。

  </p>
#+end_html
