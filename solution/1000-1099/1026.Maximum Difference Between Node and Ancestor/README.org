给定二叉树的根节点 root，找出存在于 不同 节点 A 和 B 之间的最大值
V，其中 V = |A.val - B.val|，且 A 是 B 的祖先。

#+begin_html
  <pre>
  <strong>输入：</strong>root = [8,3,10,1,6,null,14,null,null,4,7,13]
  <strong>输出：</strong>7
  <strong>解释： </strong>
  我们有大量的节点与其祖先的差值，其中一些如下：
  |8 - 3| = 5
  |3 - 7| = 4
  |8 - 1| = 7
  |10 - 13| = 3
  在所有可能的差值中，最大值 7 由 |8 - 1| = 7 得出。
