另外给定一个（包含查询指令的）数组 queries ，其中 queries[i] = [xi,
yi]。 第 i 个查询指令的答案是 nums[j] 中满足该条件的所有元素的和： xi <=
j < n 且 (j - xi) 能被 yi 整除。

#+end_html

queries.length 且 answer[i] 是第 i 个查询指令的答案对 109 + 7 取模。
