大餐 是指 恰好包含两道不同餐品 的一餐，其美味程度之和等于 2 的幂。

#+end_html

你可以搭配 任意 两道餐品做一顿大餐。

#+begin_html
  </p>
#+end_html

给你一个整数数组 deliciousness ，其中 deliciousness[i] 是第
i​​​​​​​​​​​​​​ 道餐品的美味程度，返回你可以用数组中的餐品做出的不同
大餐 的数量。结果需要对 109 + 7 取余。

  </p>
#+end_html
