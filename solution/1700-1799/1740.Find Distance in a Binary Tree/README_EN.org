#+begin_html

#+end_html

Given the root of a binary tree and two integers p and q, return the
distance between the nodes of value p and value q in the tree.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

The distance between two nodes is the number of edges on the path from
one to the other.

#+begin_html
  </p>
#+end_html

  <p>
#+end_html

 

#+begin_html
  </p>

#+begin_html
  </p>
#+end_html

#+begin_html
  <pre>
  <strong>Input:</strong> root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 0
  <strong>Output:</strong> 3
  <strong>Explanation:</strong> There are 3 edges between 5 and 0: 5-3-1-0.</pre>
