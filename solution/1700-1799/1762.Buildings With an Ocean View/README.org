表示每一个建筑物的高度。

#+end_html

#+begin_html

建筑物的右边是海洋。如果建筑物可以无障碍地看到海洋，则建筑物能看到海景。确切地说，如果一座建筑物右边的所有建筑都比它
矮 时，就认为它能看到海景。
