  <pre><strong>输入：</strong>arr = [3,3,3,3,5,5,5,2,2,7]
  <strong>输出：</strong>2
  <strong>解释：</strong>选择 {3,7} 使得结果数组为 [5,5,5,2,2]、长度为 5（原数组长度的一半）。
  大小为 2 的可行集合有 {3,5},{3,2},{5,2}。
  选择 {2,7} 是不可行的，它的结果数组为 [3,3,3,3,5,5,5]，新数组长度大于原数组的二分之一。
  </pre>
