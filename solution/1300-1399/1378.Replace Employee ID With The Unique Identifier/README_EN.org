#+begin_html
  <pre>
  Employees table:
  +----+----------+
  | id | name     |
  +----+----------+
  | 1  | Alice    |
  | 7  | Bob      |
  | 11 | Meir     |
  | 90 | Winston  |
  | 3  | Jonathan |
  +----+----------+

  EmployeeUNI table:
  +----+-----------+
  | id | unique_id |
  +----+-----------+
  | 3  | 1         |
  | 11 | 2         |
  | 90 | 3         |
  +----+-----------+

  Result table:
  +-----------+----------+
  | unique_id | name     |
  +-----------+----------+
  | null      | Alice    |
  | null      | Bob      |
  | 2         | Meir     |
  | 3         | Winston  |
  | 1         | Jonathan |
  +-----------+----------+

  Alice and Bob don&#39;t have a unique ID, We will show null instead.
  The unique ID of Meir is 2.
  The unique ID of Winston is 3.
  The unique ID of Jonathan is 1.
  </pre>
