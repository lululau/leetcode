Given an integer array nums, return the number of reverse pairs in the
array.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

A reverse pair is a pair (i, j) where 0 <= i < j < nums.length and
nums[i] > 2 * nums[j].
