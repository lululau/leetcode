Given an integer array nums of size n, return the minimum number of
moves required to make all array elements equal.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

In one move, you can increment n - 1 elements of the array by 1.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

Example 1:

#+begin_html
  </p>
#+end_html

#+begin_html
  <pre>
  <strong>Input:</strong> nums = [1,2,3]
  <strong>Output:</strong> 3
  <strong>Explanation:</strong> Only three moves are needed (remember each move increments two elements):
  [1,2,3]  =&gt;  [2,3,3]  =&gt;  [3,4,3]  =&gt;  [4,4,4]
  </pre>
