Given an integer array nums of size n, return the minimum number of
moves required to make all array elements equal.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

In one move, you can increment or decrement an element of the array by
1.

  </p>
#+end_html

  <p>
#+end_html

Example 1:

#+begin_html
  </p>
#+end_html

#+begin_html
  <pre>
  <strong>Input:</strong> nums = [1,2,3]
  <strong>Output:</strong> 2
  <strong>Explanation:</strong>
  Only two moves are needed (remember each move increments or decrements one element):
  [<u>1</u>,2,3]  =&gt;  [2,2,<u>3</u>]  =&gt;  [2,2,2]
  </pre>
