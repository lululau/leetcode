给定一个非空整数数组，找到使所有数组元素相等所需的最小移动数，其中每次移动可将选定的一个元素加1或减1。
您可以假设数组的长度最多为10000。

  [1,2,3]

  <strong>输出:</strong>
  2

  </strong>只有两个动作是必要的（记得每一步仅可使其中一个元素加1或减1）： 

  [1,2,3]  =&gt;  [2,2,3]  =&gt;  [2,2,2]
