验证原始的序列 org 是否可以从序列集 seqs 中唯一地重建。序列 org 是 1 到
n 整数的排列，其中 1 ≤ n ≤ 104。重建是指在序列集 seqs
中构建最短的公共超序列。（即使得所有
 seqs 中的序列都是该最短序列的子序列）。确定是否只可以从 seqs
重建唯一的序列，且该序列就是 org 。

  </p>
#+end_html

#+begin_html
  <pre><strong>输入：</strong>
  org: [1,2,3], seqs: [[1,2],[1,3]]

  <strong>输出：</strong>
  false

  <strong>解释：</strong>
  [1,2,3] 不是可以被重建的唯一的序列，因为 [1,3,2] 也是一个合法的序列。
  </pre>
