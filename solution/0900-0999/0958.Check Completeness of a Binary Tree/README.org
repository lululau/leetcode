#+begin_html

#+end_html

若设二叉树的深度为 h，除第 h 层外，其它各层 (1～h-1)
的结点数都达到最大个数，第 h
层所有的结点都连续集中在最左边，这就是完全二叉树。（注：第 h 层可能包含
1~ 2h 个节点。）
