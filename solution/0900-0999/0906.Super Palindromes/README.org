如果一个正整数自身是回文数，而且它也是一个回文数的平方，那么我们称这个数为超级回文数。

  </p>
#+end_html

#+begin_html

#+end_html

现在，给定两个正整数 L 和 R （以字符串形式表示），返回包含在范围 [L, R]
中的超级回文数的数目。
