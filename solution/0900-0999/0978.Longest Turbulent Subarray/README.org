当 A 的子数组 A[i], A[i+1], ...,
A[j] 满足下列条件时，我们称其为湍流子数组：

#+begin_html
  </p>
#+end_html

#+begin_html
  <ul>
#+end_html

#+begin_html
  <li>
#+end_html

若 i <= k < j，当 k 为奇数时， A[k] > A[k+1]，且当 k 为偶数时，A[k] <
A[k+1]；

#+begin_html
  </li>
#+end_html

#+begin_html
  <li>
#+end_html

或 若 i <= k < j，当 k 为偶数时，A[k] > A[k+1] ，且当 k 为奇数时， A[k]
< A[k+1]。

#+begin_html
  </li>
#+end_html

#+begin_html
  </ul>
#+end_html

#+begin_html
  <p>
#+end_html

也就是说，如果比较符号在子数组中的每个相邻元素对之间翻转，则该子数组是湍流子数组。

  </p>
#+end_html

#+begin_html
  <p>
#+end_html
