#+begin_html
  <p>
#+end_html

You are given a sorted unique integer array nums.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

Return the smallest sorted list of ranges that cover all the numbers in
the array exactly. That is, each element of nums is covered by exactly
one of the ranges, and there is no integer x such that x is in one of
the ranges but not in nums.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

Each range [a,b] in the list should be output as:

#+begin_html
  </p>
#+end_html

#+begin_html
  <ul>
#+end_html

#+begin_html
  <li>
#+end_html

"a->b" if a != b

#+begin_html
  </li>
#+end_html

#+begin_html
  <li>
#+end_html

"a" if a == b

#+begin_html
  </li>
#+end_html

#+begin_html
  </ul>
#+end_html

#+begin_html
  <p>
#+end_html

 

#+begin_html
