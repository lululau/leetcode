给你一棵 完全二叉树 的根节点 root ，求出该树的节点个数。

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

完全二叉树
的定义如下：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，并且最下面一层的节点都集中在该层最左边的若干位置。若最底层为第
h 层，则该层包含 1~ 2h 个节点。
