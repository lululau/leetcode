#+begin_html
  <pre><strong>输入：</strong>[5,6,1]
  <strong>输出：</strong>6.00000
  <strong>解释： </strong>
  以 value = 5 的节点作为子树的根节点，得到的平均值为 (5 + 6 + 1) / 3 = 4。
  以 value = 6 的节点作为子树的根节点，得到的平均值为 6 / 1 = 6。
  以 value = 1 的节点作为子树的根节点，得到的平均值为 1 / 1 = 1。
  所以答案取最大值 6。
  </pre>
