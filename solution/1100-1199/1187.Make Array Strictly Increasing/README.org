给你两个整数数组 arr1 和
arr2，返回使 arr1 严格递增所需要的最小「操作」数（可能为 0）。

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

每一步「操作」中，你可以分别从 arr1 和 arr2 中各选出一个索引，分别为 i
和 j，0 <= i < arr1.length 和 0 <= j <
arr2.length，然后进行赋值运算 arr1[i] = arr2[j]。

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

如果无法让 arr1 严格递增，请返回 -1。

#+begin_html
  </p>
#+end_html

  <p>
#+end_html
