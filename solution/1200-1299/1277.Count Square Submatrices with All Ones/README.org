1，请你统计并返回其中完全由 1 组成的 正方形 子矩阵的个数。

#+begin_html
  <pre><strong>输入：</strong>matrix =
  [
  &nbsp; [0,1,1,1],
  &nbsp; [1,1,1,1],
  &nbsp; [0,1,1,1]
  ]
  <strong>输出：</strong>15
  <strong>解释：</strong> 
  边长为 1 的正方形有 <strong>10</strong> 个。
  边长为 2 的正方形有 <strong>4</strong> 个。
  边长为 3 的正方形有 <strong>1</strong> 个。
  正方形的总数 = 10 + 4 + 1 = <strong>15</strong>.
  </pre>
#+end_html
