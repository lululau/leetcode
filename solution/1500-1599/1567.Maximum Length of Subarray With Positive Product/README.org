#+begin_html
  <pre><strong>输入：</strong>nums = [1,-2,-3,4]
  <strong>输出：</strong>4
  <strong>解释：</strong>数组本身乘积就是正数，值为 24 。
  </pre>

#+begin_html

#+begin_html
  <pre><strong>输入：</strong>nums = [0,1,-2,-3,-4]
  <strong>输出：</strong>3
  <strong>解释：</strong>最长乘积为正数的子数组为 [1,-2,-3] ，乘积为 6 。
  注意，我们不能把 0 也包括到子数组中，因为这样乘积为 0 ，不是正数。</pre>
