给定两个正整数数组 boxes 和 warehouse ，分别包含单位宽度的箱子的高度，以及仓库中
n 个房间各自的高度。仓库的房间分别从 0 到 n -
1 自左向右编号， warehouse[i] （索引从 0 开始）是第 i 个房间的高度。
