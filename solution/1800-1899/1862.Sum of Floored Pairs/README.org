给你一个整数数组 nums ，请你返回所有下标对 0 <= i, j <
nums.length 的 floor(nums[i] /
nums[j]) 结果之和。由于答案可能会很大，请你返回答案对109 +
7 取余 的结果。

#+begin_html
  </p>
#+end_html

  <p>
#+end_html

#+begin_html
  <pre><b>输入：</b>nums = [2,5,9]
  <b>输出：</b>10
  <strong>解释：</strong>
  floor(2 / 5) = floor(2 / 9) = floor(5 / 9) = 0
  floor(2 / 2) = floor(5 / 5) = floor(9 / 9) = 1
  floor(5 / 2) = 2
  floor(9 / 2) = 4
  floor(9 / 5) = 1
  我们计算每一个数对商向下取整的结果并求和得到 10 。
  </pre>
#+end_html
