for each person in the Person table, regardless if there is an address
for each of those people:

*** *SQL*
    :PROPERTIES:
    :CUSTOM_ID: sql
    :END:
#+begin_src sql
  # Write your MySQL query statement below
  SELECT p.FirstName, p.LastName, a.City, a.State
  FROM Person p
  LEFT JOIN Address a
  ON p.PersonId = a.PersonId;
#+end_src
