Write an SQL query to find all dates' id with higher temperature
compared to its previous dates (yesterday).

#+end_html

*** *SQL*
    :PROPERTIES:
    :CUSTOM_ID: sql
    :END:
#+begin_example
  select w1.Id
  from
      Weather w1,
      Weather w2
  where
      DATEDIFF(w1.RecordDate, w2.RecordDate) = 1 and w1.Temperature > w2.Temperature
#+end_example
