sum is equal to k. The same Fibonacci number can be used multiple times.

#+end_html

#+end_html

  </li>
#+end_html

#+begin_html
  <li>
#+end_html

F2 = 1

#+begin_html
  </li>
#+end_html

#+begin_html
  <li>
#+end_html

Fn = Fn-1 + Fn-2 for n > 2.

  </li>
#+end_html

#+begin_html
  </ul>
#+end_html

It is guaranteed that for the given constraints we can always find such
Fibonacci numbers that sum up to k.
