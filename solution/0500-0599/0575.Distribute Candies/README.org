给定一个偶数长度的数组，其中不同的数字代表着不同种类的糖果，每一个数字代表一个糖果。你需要把这些糖果平均分给一个弟弟和一个妹妹。返回妹妹可以获得的最大糖果的种类数。

#+begin_html
  </p>
#+end_html

  <p>
#+end_html

示例 1:

#+begin_html
  </p>
#+end_html

#+begin_html
  <pre>
  <strong>输入:</strong> candies = [1,1,2,2,3,3]
  <strong>输出:</strong> 3
  <strong>解析: </strong>一共有三种种类的糖果，每一种都有两个。
       最优分配方案：妹妹获得[1,2,3],弟弟也获得[1,2,3]。这样使妹妹获得糖果的种类数最多。
  </pre>
#+end_html
