索引从0开始长度为N的数组A，包含0到N -
1的所有整数。找到最大的集合S并返回其大小，其中 S[i] = {A[i], A[A[i]],
A[A[A[i]]], ... }且遵守以下的规则。

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

假设选择索引为i的元素A[i]为S的第一个元素，S的下一个元素应该是A[A[i]]，之后是A[A[A[i]]]...
以此类推，不断添加直到S出现重复的元素。
