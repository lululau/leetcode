假设有
n 台超级洗衣机放在同一排上。开始的时候，每台洗衣机内可能有一定量的衣服，也可能是空的。

n） 台洗衣机，与此同时将每台洗衣机的一件衣服送到相邻的一台洗衣机。

-1。
