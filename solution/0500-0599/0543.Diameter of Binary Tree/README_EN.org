Given the root of a binary tree, return the length of the diameter of
the tree.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

The diameter of a binary tree is the length of the longest path between
any two nodes in a tree. This path may or may not pass through the root.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

The length of a path between two nodes is represented by the number of
edges between them.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

 

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

Example 1:

#+begin_html
  </p>
#+end_html

#+begin_html
  <pre>
  <strong>Input:</strong> root = [1,2,3,4,5]
  <strong>Output:</strong> 3
  <strong>Explanation:</strong> 3is the length of the path [4,2,1,3] or [5,2,1,3].
  </pre>
#+end_html
