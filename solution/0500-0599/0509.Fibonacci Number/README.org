斐波那契数，通常用 F(n) 表示，形成的序列称为 斐波那契数列 。该数列由 0
和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

#+begin_html
  </p>
#+end_html

#+begin_html
  <pre>
  F(0) = 0，F(1) = 1
  F(n) = F(n - 1) + F(n - 2)，其中 n > 1
  </pre>
#+end_html

#+begin_html
  <p>
#+end_html

给你 n ，请计算 F(n) 。

#+begin_html
  </p>
#+end_html

#+end_html
