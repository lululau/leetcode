3 x 3 的幻方是一个填充有从 1 到 9 的不同数字的 3 x 3
矩阵，其中每行，每列以及两条对角线上的各数之和都相等。

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

给定一个由整数组成的 grid，其中有多少个 3 × 3 的 “幻方”
子矩阵？（每个子矩阵都是连续的）。

  </p>
#+end_html

  <pre><strong>输入: </strong>[[4,3,8,4],
        [9,5,1,9],
        [2,7,6,2]]
  <strong>输出: </strong>1
  <strong>解释: </strong>
  下面的子矩阵是一个 3 x 3 的幻方：
  438
  951
  276

  而这一个不是：
  384
  519
  762
