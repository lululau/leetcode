A 3 x 3 magic square is a 3 x 3 grid filled with distinct numbers from 1
to 9 such that each row, column, and both diagonals all have the same
sum.

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

subgrids are there?  (Each subgrid is contiguous).
