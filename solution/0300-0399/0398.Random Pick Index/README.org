给定一个可能含有重复元素的整数数组，要求随机输出给定的数字的索引。
您可以假设给定的数字一定存在于数组中。

  int[] nums = new int[] {1,2,3,3,3};
  Solution solution = new Solution(nums);

  // pick(3) 应该返回索引 2,3 或者 4。每个索引的返回概率应该相等。
  solution.pick(3);
