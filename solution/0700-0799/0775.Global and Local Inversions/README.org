数组 A 是 [0, 1, ..., N - 1] 的一种排列，N
是数组 A 的长度。全局倒置指的是 i,j 满足 0 <= i < j < N 并且 A[i] >
A[j] ，局部倒置指的是 i 满足 0 <= i < N 并且 A[i] > A[i+1] 。

  </p>
#+end_html

#+end_html
