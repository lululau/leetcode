给定一个元素都是正整数的数组A ，正整数 L 以及 R (L <= R)。

#+begin_html
  </p>
#+end_html

#+begin_html
  <p>
#+end_html

求连续、非空且其中最大元素满足大于等于L 小于等于R的子数组个数。

#+begin_html
  </p>
#+end_html

#+begin_html
  <pre><strong>例如 :</strong>
  <strong>输入:</strong> 
  A = [2, 1, 4, 3]
  L = 2
  R = 3
  <strong>输出:</strong> 3
  <strong>解释:</strong> 满足条件的子数组: [2], [2, 1], [3].
  </pre>
#+end_html
