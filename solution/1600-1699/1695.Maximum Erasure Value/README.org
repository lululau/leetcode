给你一个正整数数组 nums ，请你从中删除一个含有 若干不同元素
的子数组。删除子数组的 得分 就是子数组各元素之 和 。

  </p>
#+end_html

#+begin_html

  </p>
#+end_html

，那么它就是 a 的一个子数组。
